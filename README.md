# Telegram Bot Template for Credit Applications

This project is a sanitized template of the Telegram bot. It does not contain any real tokens, chat IDs, database files, virtual environment files, or Git history.

## Quick start
1. Create a Telegram bot via @BotFather and get a token.
2. Copy `.env.example` to `.env`.
3. Fill in `BOT_TOKEN` and `ADMIN_CHAT_ID`.
4. Create and activate a virtual environment.
5. Install dependencies from `requirements.txt`.
6. Run `python3 run.py`.

A full step-by-step guide is included in `FULL_SETUP_INSTRUCTION.md`.
